# Bloom UX Life Tracker

เว็บแอปติดตามชีวิตประจำวันภาษาไทยสำหรับ UX/UI Designer ที่อยากดูแลสุขภาพ พัฒนาทักษะ ติดตามหนี้ และสร้างรายได้เสริม

## Deploy บน Netlify

- Build command: เว้นว่าง
- Publish directory: `.`
- Entry file: `index.html`

ไฟล์ `netlify.toml` ตั้งค่า publish directory ไว้แล้ว

## Local Preview

เปิด `index.html` ได้โดยตรง หรือรัน local server:

```bash
python3 -m http.server 4173
```

แล้วเปิด `http://localhost:4173`
